Sportify is a full-stack sports equipment e-commerce web application built using **React for the frontend and Strapi CMS for the backend**. The platform allows users to browse a variety of sports products, view product details, add items to their cart, manage a wishlist, and place orders through a simple checkout process.

The application provides a smooth and responsive user experience, making it easy for customers to explore different sports categories and purchase equipment online. The backend is powered by **Strapi**, which manages products, orders, and user data through REST APIs, while the frontend built with **React and Vite** handles the interactive user interface.

Sportify demonstrates key e-commerce functionalities such as product listing, cart management, wishlist functionality, order placement, and basic admin order management. The project highlights full-stack development skills including API integration, component-based UI design, and backend content management.
